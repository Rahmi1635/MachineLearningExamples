# MachineLearningExamples

Bu depo, makine öğrenmesi dünyasına adım atarken geliştirdiğim teorik ve pratik çalışmaları içermektedir. Projeler; veri ön işlemeden model seçimine, hiperparametre optimizasyonundan başarı metriklerinin analizine kadar uçtan uca bir süreç sunar.

### 🚀 Projeler ve İçerikler
Aşağıda, repo içerisinde yer alan çalışmaların detaylarını bulabilirsiniz:

#### 1. Iris Veri Seti ile KNN Sınıflandırma

Açıklama: K-Nearest Neighbors algoritmasının temel mantığını (en yakın komşu analizi) anlatan bir giriş projesidir.

#### 2. PredictDiabetes (Diyabet Tahminleme)

Açıklama: Gerçek bir sağlık veri seti üzerinden, bir kişinin diyabet olup olmadığını tahmin eden daha gelişmiş bir sınıflandırma modelidir.

### 🛠️ Kullanılan Teknolojiler

Dil: Python

Kütüphaneler: * NumPy & Pandas (Veri manipülasyonu)

Scikit-Learn (Makine öğrenmesi algoritmaları)

Matplotlib & Seaborn (Veri görselleştirme)
